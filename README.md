# BrainVision AI — Delivery 4

Version 5.0.0 • Build 5

Delivery 4 provides the polished Streamlit application.

## Features

- MRI image upload
- EfficientNetB0 inference
- class probabilities
- Grad-CAM visualization
- PDF report generation
- session prediction history
- model-information page
- version display
- error handling
- medical-use disclaimer

## Required model files

Copy these files from the output generated by Delivery 2 into the `model/` folder:

- `brainvision_efficientnetb0_final.keras`
- `labels.json`
- `model_metadata.json`

## Local setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

On Windows:

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Community Cloud

1. Add this delivery to your GitHub repository.
2. Copy the trained model files into `model/`.
3. Push the repository to GitHub.
4. Create a Streamlit Community Cloud application.
5. Set the main file path to `app.py`.

Large model files may exceed GitHub's normal file-size limit. Use Git LFS or another model-hosting method if needed.
